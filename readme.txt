== Description ==
"Simple Easy Feedback" creates feedback in your site.
You can dynamically create simple easy feedback inputs and it is configurable from admin side.

== Installation ==
1. Download plugin and unzip it to your `/wp-content/plugins/` plugin directory.
2. Go to your plugin manager and activate it.
3. On activation, Simple Easy Feedback one configuration link is shown in plugin manager and it is also shown in admin menu.
4. Go to Simple Easy Feedback and manage form inputs that you want to display.
